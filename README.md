## About

Dimind AI is an Agentic AI Assistant built with Next.js, TypeScript and Groq AI. It automatically selects the right tool and returns clean, natural responses.

## Features

- Agentic AI Chat
- Calculator
- Weather
- Date & Time
- Dictionary
- Unit Converter
- PDF Generator
- QR Generator
- Code Explainer
- Chat History

## Tech Stack

| Category | Technologies |
|---|---|
| Frontend | Next.js, React, TypeScript |
| Styling | Tailwind CSS |
| AI | Groq API |
| Backend | Next.js API Routes |

## Installation

```bash
git clone https://github.com/RajdeepSutradhar5105/DIMIND_AI.git
cd DIMIND_AI
npm install
```

Create `.env.local`

```env
GROQ_API_KEY=YOUR_GROQ_API_KEY
MODEL_NAME=llama-3.3-70b-versatile
```

Run:

```bash
npm run dev
```
